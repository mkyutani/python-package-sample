# Python Package Sample

## Install

```
$ pip3 install .
```

## Run

```
$ python3 -m pypkgsample

Python Package Sample

$ pypkgsample
Library: /home/user1/.local/lib/python3.8/site-packages/pypkgsample/main.py
Python Package Sample
```
